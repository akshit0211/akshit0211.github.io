# Retro Resume — Old Games Theme

A single‑page, static resume website styled like an old arcade game. Includes:
- Start screen with "INSERT COIN"
- Levels (About, Skills, Projects, Experience, Contact)
- Keyboard controls (← →, M for mute, `?` for help)
- Konami Code easter egg that unlocks a Bonus Stage
- Accessible, responsive, and deployable for free

## Quick Start
Open `index.html` locally to preview.

## Customize
Search for `TODO:` in `index.html` and replace with your info:
- Name, role, bio, projects, experience, email, avatar URL, links

## Free Hosting Options
### GitHub Pages
1. Create a repo named `yourusername.github.io`.
2. Upload these files.
3. In **Settings → Pages**, set source to `main`. Your site will be live at `https://yourusername.github.io`.

### Netlify (drag & drop)
1. Go to https://app.netlify.com/drop
2. Drag the project folder; Netlify will deploy to `https://your-site-name.netlify.app`.
3. (Optional) Add a custom domain.

### Vercel
1. Import the repo at https://vercel.com/new
2. Framework preset: **Other** (static)
3. Deploy.

## Notes
- Fonts are pulled from Google Fonts (`Press Start 2P`).
- Contact form uses Netlify Forms if hosted there. Otherwise, replace with your email/mailto or a form backend service.
